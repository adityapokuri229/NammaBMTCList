Namma BMTC API Documentation

A fast, free, and completely open REST API to query Bengaluru's bus network. Our dataset covers thousands of active routes, stop sequences, and precise latitudes & longitudes.

Base URL
https://nammabmtcdb-api.onrender.com
No authentication
JSON Format
CORS Enabled
Meta
GET
/health
Simple health check. Returns {"status": "ok"} when the API is running.
Try it
https://nammabmtcdb-api.onrender.com/health
{ "status": "ok" }
GET
/stats
Returns total counts for routes, stops, and a per-category breakdown.
Try it
https://nammabmtcdb-api.onrender.com/stats
{ "total_routes": 2957, "total_stops": 4927, "by_category": { "mf": 124, "airport": 31, "radial": 34, "orbital": 689, "express": 12, "general": 2067 } }
Routes
GET
/routes
Paginated list of all routes. Filter by category.
Query Parameters
category
optional
string
Filter by category: mf airport radial orbital express general
page
optional
integer
Page number (default: 1)
limit
optional
integer
Results per page. Max 200 (default: 50)
Try it
category page limit
https://nammabmtcdb-api.onrender.com/routes?page=1&limit=10
GET
/routes/search
Search routes by route number or name. Dashes and spaces are stripped before matching, so 500D finds 500-D and kia8 finds KIA-8.
Query Parameters
q
required
string
Search keyword — matches route ID or route name
category
optional
string
Narrow to a specific category
limit
optional
integer
Max results (default: 20, max: 100)
Try it
query (q) limit
https://nammabmtcdb-api.onrender.com/routes/search?q=Hebbal&limit=5
{ "query": "500-D", "total": 1, "routes": [ { "route_id": "500-D", "name": "Hebbal ⇔ Central Silk Board", "category": "orbital", "stops_count": 36 } ] }
GET
/routes/between
Find all routes that pass through both stops in the correct order. Useful for journey planning.
Query Parameters
from
required
string
Origin stop name (case-insensitive)
to
required
string
Destination stop name (case-insensitive)
Try it — pick stops from the actual dataset
from stop
to stop
https://nammabmtcdb-api.onrender.com/routes/between?from=Kempegowda%20Bus%20Station&to=Central%20Silk%20Board
{ "from": "Kempegowda Bus Station", "to": "Central Silk Board", "total": 4, "routes": [ { "route_id": "500-D", "name": "Hebbal ⇔ Central Silk Board", "category": "orbital", "from_stop_seq": 8, "to_stop_seq": 36, "stops_between": 27 } ] }

/stops/nearby
Find all bus stops within a radius of a coordinate, sorted by distance. Great for location-aware apps.

required
number
Latitude (decimal degrees)
lng
required
number
Longitude (decimal degrees)
radius
optional
number
Search radius in metres. Default: 500. Max: 5000.
limit
optional
integer
Max results (default: 20)
Try it — or

GET
/stops/{stop_name}
Get coordinates for a specific stop by name (case-insensitive). The stop name goes in the URL path — spaces are encoded as %20.
Try it
stop_name
https://nammabmtcdb-api.onrender.com/stops/Kempegowda%20Bus%20Station
GET
/stops/{stop_name}/routes
All routes that pass through a given stop, with stop sequence number. One of the most useful endpoints.
