Namma BMTC API Documentation

A free, open REST API for Bengaluru's bus network. Query 2,957 routes, 4,927 stops, and precise geo-coordinates. No auth required.

Base URL
https://nammabmtcdb-api.onrender.com
No auth required
JSON responses
CORS enabled
Meta
GET
/stats
Returns total counts for routes, stops, and a per-category breakdown.
Try it
https://nammabmtcdb-api.onrender.com/stats
{ "total_routes": 2957, "total_stops": 4927, "routes_with_variants": 780, "by_category": { "mf": 124, "airport": 31, "radial": 34, "orbital": 689, "express": 12, "general": 2067 } }

GET
/routes
Paginated list of all routes. Filter by category.
Query Parameters
category
optional
string
Filter by category: mf airport radial orbital express general
page
optional
integer
Page number (default: 1)
limit
optional
integer
Results per page. Max 200 (default: 50)
Try it
category page limit
https://nammabmtcdb-api.onrender.com/routes?page=1&limit=10
GET
/routes/search
Search routes by route number or name. Matches "500-D", "Majestic", "Hebbal", etc.
Query Parameters
q
required
string
Search keyword — matches route ID or route name
category
optional
string
Narrow to a specific category
limit
optional
integer
Max results (default: 20, max: 100)
Try it
query (q) limit
https://nammabmtcdb-api.onrender.com/routes/search?q=Hebbal&limit=5

/routes/between
Find all routes that pass through both stops in the correct order. Useful for journey planning.

Exact name of the origin stop

string

integer
