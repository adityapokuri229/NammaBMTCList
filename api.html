Developer API Documentation

A fast, free, and completely open REST API to query Bengaluru's bus network. Our dataset covers thousands of active routes, stop sequences, and precise latitudes & longitudes.

Base URL
https://nammabmtcdb-api.onrender.com
GET /stats
Retrieve system-wide metadata, tracking the total number of routes, stops, and variants currently active in the database.
$curl -X GET "https://nammabmtcdb-api.onrender.com/stats" { "total_routes": 2104, "total_stops": 8426, "by_category": { "mf": 58, "airport": 17, "general": 1605 } }
GET /routes/search
Search for bus routes using a query string. Matches against route numbers (e.g. "500-D") and route names (e.g. "Hebbal").
Query Parameters
q
required		 string The search keyword to look for.
category string Filter by category ID (e.g., airport, mf, orbital).
$curl -X GET "https://nammabmtcdb-api.onrender.com/routes/search?q=500&limit=1" { "query": "500", "total": 1, "routes": [ { "route_id": "500-D", "name": "Hebbal to Central Silk Board", "category": "orbital", "stops_count": 36, "variants_count": 2 } ] }
GET /routes/{route_id}/stops
Get the precise sequence of stops for a specific route variant, including exact geographical coordinates.
Path Parameters
route_id
required		 string The exact route ID (e.g. 500-D).
$curl -X GET "https://nammabmtcdb-api.onrender.com/routes/MF-1A/stops" { "route_id": "MF-1A", "variant": 0, "name": "Swami Vivekananda Road Metro Station ⇔ Mahadevapura", "stops": [ { "sequence": 1, "name": "Swami Vivekananda Road Metro Station", "lat": 12.985655, "lng": 77.644919 }, { "sequence": 2, "name": "Isolation Hospital", "lat": 12.986684, "lng": 77.646549 } ] }
GET /stops/nearby
Retrieve all bus stops near a specific geographical coordinate, sorted by distance.
Query Parameters
lat
required		 number Latitude of the search center.
lng
required		 number Longitude of the search center.
radius number Search radius in meters (Default: 500, Max: 5000).
$curl -X GET "https://nammabmtcdb-api.onrender.com/stops/nearby?lat=12.9716&lng=77.5946&radius=200" { "lat": 12.9716, "lng": 77.5946, "radius_m": 200, "total": 2, "stops": [ { "name": "M.A.C.C.O.Society", "lat": 12.972306, "lng": 77.593683, "distance_m": 126 }, { "name": "St.Marthas Hospital Corporate", "lat": 12.971168, "lng": 77.592965, "distance_m": 184 } ] }
